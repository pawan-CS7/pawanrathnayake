                  skills, projects, and services with a working contact form.

-----------------------------------------------------------------------------

TECHNOLOGIES USED:
------------------
1. HTML5        : For the structure of the website.
2. CSS3         : For styling, layout (Flexbox/Grid), and animations.
3. JavaScript   : For interactive elements and logic.
4. Libraries    :
   - ScrollReveal.js : For scroll animations.
   - Typed.js        : For the typing text effect in the home section.
   - Boxicons        : For social media and UI icons.
5. EmailJS      : For the functional contact form (sends emails directly).

HOW TO RUN:
-----------
1. Unzip the project folder.
2. Ensure you have an active internet connection (required for Icons, 
   ScrollReveal, Typed.js, and EmailJS libraries to load via CDN).
3. Double-click on 'index.html' to open it in your default web browser.
4. To test the contact form, fill in the details and click "Submit".
